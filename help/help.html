Feeder! Feeds Reader

General

Bienvenido a Feeder, un lector de noticias social, sencillo y rápido que te ayudará a mantenerte informado allí donde te encuentres.

Tras haberte registrado, lo primero que te pediremos es que crees tu planeta, un planeta sirve para organizar tus feeds de forma cómoda, así puede tener un planeta de tecnología, recetas de cocina o lo que se te ocurra.

Los datos necesarios para crear el planeta son un nombre y una breve descripción del mismo. Así por ejemplo, puedes tener un planeta llamado Linux con una descripción Noticias sobre linux. Esto se completa en la siguiente pantalla.

¡Ya tienes un planeta creado! Es momento de llenarlo de habitantes. Para ello, creamos una nueva suscripción en la pantalla que acaba de aparecer. Una suscripción se genera a partir de un canal, y es la dirección a un archivo xml que contiene toda la informcación de las noticias como el título del artículo, el contenido propio, la fecha de creación, etc.

Un buen canal de noticias es Barrapunto. Vamos a suscribirnos a las noticias de esta página.

Ya tenemos pobladores en nuesto planeta como muestra la siguiente imagen.

Podemos crear tantos planetas como queramos y dentro de cada uno, todas las suscripciones que nos interesen para estar siempre informados.

¿Y ahora qué?

Ahora es momento de administrar tus suscripciones y tus planetas. Para ello desde tu página principal tienes dos enlaces suscripciones y eliminar. El enlace a eliminar borrará el planeta de la base de datos. El enlace suscripciones mostrará una lista de los canales contenidos en el planeta con las opciones de eliminarlo o suscribirte a las actualizaciones del planeta.

Más Cosas

Feeder! también cuenta con capacidad para almacenar tu perfil. A este menú puedes acceder desde la barra de navegación situada en la esquina superior derecha de la aplicación. Dispone del sistema breadcrumb para saber en todo momento el camino que has recorrido dentro de la aplicación. Feeder! dispone de generación de rss para cada planeta.

El sistema de actualización de Feeder! se realiza mediante una tarea de cron, el demonio de tareas periódicas de Linux, que mediante un script de bash actualiza todos los feeds de la base de datos. Para que no sea una carga excesiva de trabajo, en nuestra base de datos sólo existe un canal y todos los planetas lo comparten.

Feeder! es Software Libre, el código fuente lo podeis encontrar en el sitio web GitHub, se publica bajo licencia GPL.
